本页直接引用 shadertoy.com/media/a/ 的图片 — 在浏览器中打开即可一键保存全部资源

数据来源： Shadertoy Unofficial — Shadertoy media files（by fabriceN, 2019-07-23, updated 2026-02-06）

如何一次性下载全部纹理：

1. 用系统浏览器（Chrome / Edge / Firefox）打开本 HTML 文件
2. 等待页面中所有图片加载完成（图片来自 shadertoy.com，需要能访问该网站）
3. 按 Ctrl+S（或菜单 → 另存为），保存类型选择 「网页，全部」
4. 浏览器会自动将全部图片保存到一个 _files 文件夹中
5. 将该文件夹中所有 .jpg / .png 文件复制到项目的 assets/media/a/ 目录即可

原理：本页的所有图片 <img src> 直接指向 https://www.shadertoy.com/media/a/{hash}.ext， 浏览器加载页面时会自动下载这些图片到缓存，Ctrl+S 保存网页时会一并导出。
ShaderToy JSON 中引用的 /media/a/{hash}.ext 路径，程序会自动映射到 assets/media/a/ 查找。
