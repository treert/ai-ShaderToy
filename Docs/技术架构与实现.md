# 技术架构与实现

## 技术栈
| 项目 | 选型 |
|------|------|
| 语言 | C++17 |
| 构建 | CMake 3.16+，FetchContent 自动拉取依赖 |
| 窗口/事件 | SDL2 (release-2.30.12, 静态链接) |
| OpenGL 加载 | GLAD (v0.1.36, OpenGL 3.3 Core) |
| 桌面壁纸嵌入 | Win32 API (WorkerW 方案) |
| 编译器 | MSVC (Visual Studio 2022) |

## 源码结构
```
src/
├── main.cpp                # 程序入口、事件循环、命令行解析
├── renderer.h/.cpp         # OpenGL 渲染器（全屏四边形 + uniform 传递）
├── shader_manager.h/.cpp   # ShaderToy 着色器加载、包装、编译
└── wallpaper.h/.cpp        # Win32 桌面壁纸嵌入（WorkerW）

assets/
└── shaders/
    └── default.glsl        # 默认示例着色器（彩色渐变动画）
```

## 构建与运行
```bash
# 配置（首次会自动下载 SDL2 和 GLAD）
cmake -B build

# 编译
cmake --build build --config Release

# 运行 — 窗口模式
build\Release\ShaderToyDesktop.exe

# 运行 — 壁纸模式
build\Release\ShaderToyDesktop.exe --wallpaper

# 运行 — 指定自定义 shader
build\Release\ShaderToyDesktop.exe --shader assets\shaders\default.glsl
```

## 命令行参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--shader <path>` | ShaderToy GLSL 文件路径 | `assets/shaders/default.glsl` |
| `--wallpaper` | 启用桌面壁纸模式 | 关闭 |
| `--width <n>` | 窗口宽度（壁纸模式下自动全屏） | 800 |
| `--height <n>` | 窗口高度（壁纸模式下自动全屏） | 600 |
| `--fps <n>` | 目标帧率 | 60 |
| `--help` | 显示帮助信息 | — |

## 渲染流程
```
用户 shader (mainImage)
    → ShaderManager 包装为完整 Fragment Shader
    → OpenGL 编译链接
    → Renderer 每帧设置 uniform 并绘制全屏四边形
    → SDL swap buffer 显示
```

## ShaderToy 兼容

用户只需提供 ShaderToy 格式的 `mainImage(out vec4, in vec2)` 函数，
`ShaderManager` 会自动包装为完整的 Fragment Shader：
```glsl
#version 330 core
uniform vec3      iResolution;
uniform float     iTime;
uniform float     iTimeDelta;
uniform int       iFrame;
uniform vec4      iMouse;
uniform vec4      iDate;
uniform float     iSampleRate;

out vec4 _fragColor_out;

// ← 用户的 mainImage 代码插入此处 →

void main() {
    _fragColor_out = vec4(0.0);
    mainImage(_fragColor_out, gl_FragCoord.xy);
}
```

### ShaderToy Uniform 支持状态
| 变量 | 类型 | 状态 | 说明 |
|------|------|------|------|
| `iResolution` | `vec3` | ✅ 已接入 | 视口分辨率 (px) |
| `iTime` | `float` | ✅ 已接入 | 播放时间 (秒) |
| `iTimeDelta` | `float` | ✅ 已接入 | 帧间隔时间 (秒) |
| `iFrame` | `int` | ✅ 已接入 | 帧计数 |
| `iMouse` | `vec4` | ✅ 已接入 | xy=当前位置, zw=按下位置(松开取负) |
| `iDate` | `vec4` | ✅ 已接入 | 年/月(0-11)/日/当天秒数 |
| `iSampleRate` | `float` | ⬜ 已声明 | 音频采样率（待接入） |
| `iChannel0~3` | `sampler2D` | ⬜ 未实现 | 输入纹理通道 |
| `iChannelResolution` | `vec3[4]` | ⬜ 未实现 | 各通道分辨率 |

## 壁纸嵌入原理

通过 Win32 API 向 `Progman` 发送未公开消息 `0x052C`，Windows 会创建 `WorkerW` 窗口层。
将渲染窗口 `SetParent` 到该层，即可显示在桌面图标之下，实现动态壁纸效果。

```
Desktop (Progman)
  └─ SHELLDLL_DefView (图标列表)
      └─ SysListView32

→ 发送 0x052C 后产生 WorkerW
→ SetParent(渲染窗口, WorkerW)
→ 渲染窗口显示在图标下方
```
